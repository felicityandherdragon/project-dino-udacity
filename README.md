# Dinosaurs Infographic Project
How do you compare to dinosaurs?

## To run
### Clone the repo
```
git clone https://github.com/felicityandherdragon/project-dino-udacity.git
```

### Install npm packages and start the app locally
```
npm install
npm run start:dev
```
